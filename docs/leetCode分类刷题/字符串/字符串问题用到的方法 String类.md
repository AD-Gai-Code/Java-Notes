####  跟字符串有关的其它类的方法
`Integer.parseInt`用法：

`parseInt(String s)`: 返回用十进制参数表示的整数值。
`parseInt(int i)`: 使用指定基数的字符串参数表示的整数 (基数可以是 10, 2, 8, 或 16 等进制数) 。
